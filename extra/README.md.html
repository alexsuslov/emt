Версия для работы в браузере

Simple

  • работают правила не изменяющие тэги
  • как jQuery плагин

Использование:

<textarea name="emt" rows="10" class='emt col-lg-12 col-xs-12'></textarea>
<p class='emt'></p>

$(function() {
  var $p = $('p.emt');
  var emt = $p.emt();
  $('textarea.emt').on('keyup' ,function(e){
    $p.html( e.target.value);
    emt = $('p.emt').emt();
  });
});
