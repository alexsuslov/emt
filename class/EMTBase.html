# (void) set(self, selector, key, value)

Установить настройки для тертов и правил

  1. если селектор является массивом, то тогда утсановка правил будет выполнена для каждого элемента этого массива, как отдельного селектора.
  2. Если $key не является массивом, то эта настрока будет проставлена согласно селектору
  3. Если $key массив
  4. то будет задана группа настроек
  5. если $value массив , то настройки определяются по ключам из массива $key, а значения из $value
  6. иначе, $key содержит ключ-значение как массив @param mixed $selector @param mixed $key @param mixed $value
